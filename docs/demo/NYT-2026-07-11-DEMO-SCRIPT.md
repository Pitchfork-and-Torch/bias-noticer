# Bias Noticer — Demo video script & storyboard

**Article (example / narrative target):**  
https://www.nytimes.com/2026/07/11/business/media/new-york-times-trump-subpoenas.html  

**Theme:** Media literacy on *self-referential* coverage (outlet covering subpoenas / bias-related legal pressure).  
**Tone:** Neutral, empowering, non-partisan. Celebrate critical reading — not team scoring.  
**Runtime:** 75 seconds (±15).  
**Formats:** 9:16 vertical (primary) + 16:9 horizontal cut.

---

## Ethics slate (mandatory 2s open **and** soft recap on end card)

**Asset:** `assets/00-ethics-slate.jpg` + `storyboard/SB05-end-card.jpg`

> Bias Noticer analyzes rhetorical techniques. It does not declare legal truth.  
> “Reader extract” only reformats text already loaded in your browser — not a paywall crack, not a subscription substitute.

**VO must never say:** “debunked,” “destroyed,” “proves bias lawsuit,” or party victory language.

**[1:02–1:15]**  
“Customize sensitivity, export a report, keep your API key local. See through the propaganda — and keep your mind free. Bias Noticer.”

## Production recipe

1. **Capture:** Windows Game Bar / OBS → 1440×2560 (vertical) or 1920×1080.  
2. **Browse:** Dedicated profile with extension loaded; use Demo if API credits low, then real Grok when available.  
3. **Edit:** CapCut / Descript — punch-in zooms on tooltip & gauge; 0.3s ease transitions.  
4. **VO:** Descript Studio Sound or ElevenLabs; 145–155 wpm.  
5. **Music:** Low bed, no lyrics (news-safe).  
6. **Captions:** Burned-in large captions for silent feed scroll.

## Narrative insight (for creators)

Self-referential legal/media stories are ideal demos because:

1. Stakeholders have incentives → framing is likely.  
2. Primary sources (dockets, filings) exist for counter-checks.  
3. Audiences of all politics care about press power — keep the tool technique-first so nobody can honestly call it a team jersey.
