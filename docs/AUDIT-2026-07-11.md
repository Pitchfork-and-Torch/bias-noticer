# Bias Noticer — Extension audit (2026-07-11)

## Scope

Code review of MV3 extension at `bias-noticer` (popup, side panel, options, content, background, lib). Legal stance: no paywall bypass; DOM-only reader + research paste only.

## Findings (severity)

### Critical (fixed in 1.4.0)

| Issue | Fix |
|-------|-----|
| Content script re-inject registered duplicate listeners | Guard `__BN_CONTENT__` + `PING` before inject |
| Version drift (1.2 / 1.3.0 / 1.3.x in UI vs package) | `lib/version.ts` + package.json; drop hardcoded WXT version |
| Sender tab = options tab broke analyze | `resolveTargetTabId` + optional `tabId` (1.3.1+) |
| Side panel unusable UX on meters | Thin-page analysis, status banner, Research tab (1.3.2–1.4.0) |

### High (fixed)

| Issue | Fix |
|-------|-----|
| API hang with no timeout | AbortController 90–120s |
| API errors could echo secrets | `redactSecrets` |
| Grok spent on empty paywall shells | `isTooThinForModel` → structured guidance analysis |
| Fragile JSON parse from free-form chat | Prefer `response_format: json_object` with fallback |
| Theme logic triplicated | `lib/theme.ts` |

### Medium (fixed / improved)

| Issue | Fix |
|-------|-----|
| Weak paywall detection strings | Expanded `PAYWALL_HINTS` |
| Sparse NYT-style bodies | `extractStoryModules` selectors |
| Panel stale when switching tabs | visibility + `tabs.onActivated` + poll refresh |
| Test profiles risk of commit | `.gitignore` entries |
| `getActiveTab` wrong window | Prefer last-focused http(s) |

### Accepted / deferred

| Item | Notes |
|------|--------|
| No automated unit tests | Smoke scripts under `scripts/`; add Vitest later |
| Host permissions broad (`http(s)://*/*`) | Required for inject/highlight; document in store listing |
| Service worker memory cache | Tab analysis map is in-memory; lost on SW sleep — cache storage covers re-scan |
| i18n incomplete | `docs/I18N.md` scaffold only |
| Firefox path | `build:firefox` exists; not primary test target |
| Archive auto-fetch | Intentionally **not** implemented (ethics/ToS) |

## Architecture snapshot

```
popup / sidepanel / options  →  runtime messages  →  background (SW)
                                                      ├─ storage (settings, cache)
                                                      ├─ xAI chat (BYOK)
                                                      └─ tabs messaging → content script
                                                                            ├─ extract / reader
                                                                            └─ highlights / tooltips
```
